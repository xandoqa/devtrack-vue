# DevTrack 

Projeto desenvolvido com Vue.js 3 para gerenciamento de estudos e projetos.

##  Tecnologias
- Vue 3
- Vite
- Pinia
- Vue Router
- Axios

##  Como rodar o projeto

### 1. Instalar dependências
```bash
npm install
```

### 2. Rodar o servidor
```bash
npm run dev
```

### 3. Acessar
Acesse o LocalHost

## 📁 Estrutura
- src/views → páginas
- src/router → rotas
- src/store → estado global (expandível)

## 💡 Objetivo
Projeto para portfólio com foco em boas práticas de frontend moderno.
